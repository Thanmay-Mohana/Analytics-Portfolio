# Teen Mental Health & Social Media Usage Dashboard

## Project Overview

This Power BI dashboard explores the relationship between social media usage, lifestyle habits, stress, anxiety, sleep, physical activity, academic performance, and depression risk among teenagers.

This project was created as part of **The Insight Stack – Week 8**, focusing on interactive storytelling features in Power BI such as drill-through, report tooltips, bookmarks, buttons, slicers, page navigation, and cross-filtering.

## Business Question

How can a school wellbeing team explore the relationship between social media usage, lifestyle habits, stress, anxiety, and depression risk among teenagers?

## Important Disclaimer

This dataset is synthetic and created for educational/dashboard demonstration purposes only. It should not be used for clinical diagnosis, healthcare decisions, or medical advice.

## Dashboard Pages

### Page 1: Executive Overview

This page provides a high-level summary of the dataset.

Key visuals include:

- Total teenagers
- Depression rate
- Average social media hours
- Average sleep hours
- Average stress level
- Depression status breakdown
- Depression rate by gender
- Depression rate by preferred platform
- Average indicators by depression status

Interactive features used:

- Slicers
- Page navigator
- Reset filters button
- Cross-filtering

### Page 2: Lifestyle Deep Dive

This page explores how lifestyle and behavioural indicators differ between depressed and non-depressed teenagers.

Key visuals include:

- Social media hours by depression status
- Sleep hours by depression status
- Stress and anxiety by depression status
- Physical activity by depression status
- Academic performance by depression status
- Social media vs sleep scatter chart

Interactive features used:

- Report tooltips
- Bookmark toggle
- Bookmark navigator
- Slicers
- Cross-filtering

### Page 3: Segment Detail

This page provides a detailed drill-through view for selected segments.

Users can drill through by:

- Gender
- Preferred platform
- Age group
- Depression status

Key visuals include:

- Segment KPIs
- Age distribution
- Platform breakdown
- Mental health indicators
- Row-level detail table

Interactive features used:

- Drill-through
- Back button
- Segment-level filtering
- Detail table
